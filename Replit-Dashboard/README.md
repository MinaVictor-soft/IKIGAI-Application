# Replit-Dashboard

Admin Dashboard deployment for Replit environment.

## Purpose
This folder contains the admin dashboard code optimized for Replit deployment. Any changes made to the main `admin-dashboard/` folder should be synced here for deployment.

## Sync Instructions
1. Make changes in `admin-dashboard/` (main code)
2. Update this folder with the same changes
3. Push to `Replit-Dashboard` branch via git

## Deployment
Push to Replit using: `git push origin Replit-Dashboard`
