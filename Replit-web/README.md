# Replit-web

Web Application deployment for Replit environment.

## Purpose
This folder contains the web app code optimized for Replit deployment. Any changes made to the main `web-app/` folder should be synced here for deployment.

## Sync Instructions
1. Make changes in `web-app/` (main code)
2. Update this folder with the same changes
3. Push to `Replit-web` branch via git

## Deployment
Push to Replit using: `git push origin Replit-web`
