## 📤 How to Push to GitHub

### Step 1: Create a Repository on GitHub

1. Go to **https://github.com/new**
2. Repository name: `IKIGAI-Quest` (or your preferred name)
3. Description: `Complete gamification platform with web, mobile, admin dashboard and backend API`
4. Choose: **Private** or **Public**
5. Click **Create repository**

### Step 2: Add Remote and Push

After creating the repository, GitHub will show you commands. Use one of these:

**HTTPS Method** (easiest):
```bash
git remote add origin https://github.com/YOUR-USERNAME/IKIGAI-Quest.git
git branch -M main
git push -u origin main
```

**SSH Method** (requires SSH key setup):
```bash
git remote add origin git@github.com:YOUR-USERNAME/IKIGAI-Quest.git
git branch -M main
git push -u origin main
```

### Step 3: Verify Push

After pushing, run:
```bash
git remote -v
```

You should see:
```
origin  https://github.com/YOUR-USERNAME/IKIGAI-Quest.git (fetch)
origin  https://github.com/YOUR-USERNAME/IKIGAI-Quest.git (push)
```

## 🔐 Setup Authentication (for HTTPS)

### Option A: Personal Access Token (Recommended)

1. Go to: Settings → Developer settings → Personal access tokens
2. Click "Tokens (classic)" → "Generate new token"
3. Give it permission: `repo` (full control of private repositories)
4. Copy the token
5. When Git prompts for password, paste the token

### Option B: SSH Keys (More secure)

1. Generate SSH key:
```bash
ssh-keygen -t ed25519 -C "your-email@example.com"
```

2. Add to SSH agent:
```bash
eval "$(ssh-agent -s)"
ssh-add ~/.ssh/id_ed25519
```

3. Add public key to GitHub:
   - Settings → SSH and GPG keys → New SSH key
   - Paste content of `~/.ssh/id_ed25519.pub`

## 🚦 After First Push

### Set up GitHub Actions CI/CD (Optional)

Create `.github/workflows/build.yml`:
```yaml
name: Build & Test
on: [push, pull_request]
jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: actions/setup-node@v3
        with:
          node-version: '18'
      - run: npm ci
      - run: npm run build
      - run: npm run lint
```

### Clone on Another Machine

```bash
git clone https://github.com/YOUR-USERNAME/IKIGAI-Quest.git
cd IKIGAI-Quest

# Install dependencies
npm install
cd backend && npm install && cd ..
cd web-app && npm install && cd ..
cd admin-dashboard && npm install && cd ..

npm run dev
```
