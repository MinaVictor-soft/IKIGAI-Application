# Replit-Backend

Backend API deployment for Replit environment.

## Purpose
This folder contains the backend code optimized for Replit deployment. Any changes made to the main `backend/` folder should be synced here for deployment.

## Sync Instructions
1. Make changes in `backend/` (main code)
2. Update this folder with the same changes
3. Push to `Replit-Backend` branch via git

## Deployment
Push to Replit using: `git push origin Replit-Backend`
